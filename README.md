# zk-SNARKs from Scratch in Rust

This repository contains a step-by-step implementation of zk-SNARKs (Zero-Knowledge Succinct Non-Interactive Argument of Knowledge) protocols, built from scratch in Rust.

The goal of this project is to understand the underlying mathematics of Zero-Knowledge Proofs by implementing primitive operations without relying on high-level ZK libraries.

このリポジトリは、zk-SNARKs（ゼロ知識証明プロトコルの一種）を Rust でゼロから実装したものです。 

既存のハイレベルな ZK ライブラリに依存せず、基本的な演算処理（プリミティブ）から自作することで、ゼロ知識証明の背後にある数学的仕組みを深く理解することを目的としています。
